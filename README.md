Certamen N°2 - Frontend

Sitio web estático para la cafetería ficticia "Café Universitario", un
emprendimiento dedicado a la venta de café, pastelería y productos artesanales.

La página presenta la información de la empresa, muestra sus productos y permite
registrar solicitudes de información de los clientes mediante un formulario.
Cada solicitud enviada se muestra de inmediato en una tabla dentro de la misma
página, los registros se mantienen mientras la página permanece abierta.

Secciones
- Header con menú de navegación
- Banner principal
- Nosotros
- Productos (tres productos con imagen, nombre, descripción y precio)
- Solicitud de Información (formulario)
- Solicitudes Registradas (tabla)
- Footer

Tecnologías
- HTML5, Para la estructura semántica del contenido.
- CSS3, Para la estilización del sitio. 
- JavaScript, Para otorgarle la funcionalidad y manejo del DOM.
- Git/GitHub: Para el control de versiones y gestión del historial de cambios.

Para visualizar y ejecutar este proyecto correctamente en un entorno local:

- Clonar el repositorio: Mediante git clone https://github.com/camarada92/Certamen2-CafeUniversitario.git

- Abrir el proyecto: Navega a la carpeta del proyecto y abre el archivo index.html en tu navegador web preferido.
